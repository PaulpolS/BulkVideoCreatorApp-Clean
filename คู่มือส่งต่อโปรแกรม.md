# คู่มือติดตั้งและใช้งาน CreatorHub Production Suite

CreatorHub Production Suite เป็นเว็บแอปสำหรับช่วยทำงานคอนเทนต์แบบหลายขั้นตอนในที่เดียว ตั้งแต่หาไอเดีย วิเคราะห์คู่แข่ง สร้างบทความ/โพสต์ ทำ stock ลงเพจ สร้าง prompt ภาพและวิดีโอ ไปจนถึงจัดการ assets และช่วยงานตัดต่อ

## วิธีติดตั้งแบบใช้คำสั่ง

ต้องมี Node.js ติดตั้งในเครื่องก่อน แนะนำ Node.js เวอร์ชัน LTS

```bash
git clone <ใส่ลิงก์ GitHub repo ตรงนี้>
cd BulkVideoCreatorApp-Clean
npm install
npm run dev
```

หลังรันคำสั่งสุดท้าย ให้เปิด URL ที่ Terminal แสดง เช่น

```text
http://localhost:5173/
```

ถ้าพอร์ต `5173` ถูกใช้อยู่แล้ว Vite จะเลื่อนไปพอร์ตถัดไป เช่น `5174`, `5175`

## วิธีติดตั้งผ่าน GitHub Desktop

1. เปิด GitHub Desktop
2. กด `File` > `Clone Repository`
3. เลือก repo นี้ หรือวาง URL ของ repo
4. เลือกโฟลเดอร์ที่จะเก็บโปรเจกต์ แล้วกด `Clone`
5. เปิดโฟลเดอร์โปรเจกต์ใน Terminal
6. รันคำสั่งนี้:

```bash
npm install
npm run dev
```

จากนั้นเปิด URL ที่ Terminal แสดง

## การตั้งค่าที่ต้องทำใหม่ในเครื่องคนใช้

โปรไฟล์ API และ token ต่าง ๆ ไม่ได้ส่งติดไปกับ Git เพื่อความปลอดภัย ผู้ใช้แต่ละคนต้องตั้งค่าเองที่ปุ่ม `ตั้งค่า API (คีย์)` มุมบนของแอป

คีย์ที่อาจต้องใช้ตามฟีเจอร์:

- OpenRouter API Key: ใช้กับการวิเคราะห์ด้วย AI, สร้างสมอง, เขียนโพสต์, สร้าง prompt
- Dropbox Token หรือ Dropbox App Key/Secret: ใช้ดึงรูปจาก Dropbox และสร้างลิงก์รูป
- Kie.ai API Key: ใช้กับบางงานสร้างภาพ/เสียง
- Apify API Key: ใช้ดึงข้อมูลโซเชียลและวิเคราะห์คู่แข่ง
- Google API Key: ใช้กับ dashboard หรือข้อมูลจาก Google Sheets

## คำสั่งที่ใช้บ่อย

รันแอปสำหรับใช้งาน/พัฒนา:

```bash
npm run dev
```

ทดสอบ build ว่าโปรเจกต์ไม่พัง:

```bash
npm run build
```

ดู build production หลัง build:

```bash
npm run preview
```

อัปเดตโปรเจกต์หลังมีการแก้ไขจาก GitHub:

```bash
git pull
npm install
npm run dev
```

## โปรแกรมนี้ทำอะไรได้บ้าง

### 1. สร้างและจัดการคอนเทนต์ลงเพจ

- สร้างบทความและโพสต์ด้วย AI
- สร้าง prompt สำหรับภาพประกอบ
- ทำงานแบบทีละชิ้นหรือหลายรายการ
- เก็บ stock บทความและคอนเทนต์ไว้ใช้ซ้ำ
- ใช้ระบบสมอง/Prompt Brain เพื่อวิเคราะห์ CSV ตัวอย่าง แล้วเขียนให้ตรงสไตล์เดิมของเพจ

### 2. ทำ Stock ลงเพจ

- ดึงหัวข้อหรือข้อมูลจากไฟล์ CSV
- วิเคราะห์ตัวอย่างโพสต์เดิมเพื่อสร้างสมองของเพจ
- แบ่ง CSV ยาว ๆ เป็นหลายส่วนให้ AI วิเคราะห์ทีละชุด
- เขียนโพสต์จากรูปใน Dropbox โดยยึด pattern จาก CSV
- Export ผลลัพธ์เป็น CSV สำหรับนำไปใช้ต่อ

### 3. ค้นหาไอเดียคอนเทนต์และข่าว

- ดึงข่าวหรือบทความจากแหล่งข่าว/RSS
- สรุปข่าวให้อ่านง่าย
- ให้ AI ช่วยคัดไอเดียที่น่าสนใจ
- ส่งไอเดียไปเก็บในคลังบทความหรือใช้ต่อกับ workflow อื่น

### 4. วิเคราะห์คู่แข่งและเรดาร์คอนเทนต์

- เก็บรายชื่อเพจ/ช่อง/บัญชีคู่แข่ง
- ดึงข้อมูลโพสต์หรือคอนเทนต์ผ่าน API ที่ตั้งค่าไว้
- วิเคราะห์โพสต์ที่น่าสนใจ
- ใช้เป็นข้อมูลตั้งต้นสำหรับหาแนวคอนเทนต์ใหม่

### 5. โคลนนิ่งสไตล์เพจ/ภาพ

- วิเคราะห์ภาพตัวอย่างเพื่อจับสไตล์ สี อารมณ์ องค์ประกอบ และ layout
- สร้าง prompt ใหม่ให้คงแนวภาพเดิม
- ใช้ช่วยทำภาพแนวเดียวกันสำหรับหลายโพสต์

### 6. Video Prompt Generator

- ช่วยสร้าง prompt สำหรับวิดีโอ
- แบ่งไอเดียเป็น scene
- วางรายละเอียดภาพ มุมกล้อง อารมณ์ แสง และ continuity
- ใช้ต่อกับเครื่องมือสร้างวิดีโอ AI ได้

### 7. Bulk Video Creator และงานตัดต่อ

- จัดการงานสร้างวิดีโอหลายรายการ
- ใช้ content stock เป็นวัตถุดิบ
- ตั้งค่า template วิดีโอ ฟอนต์ สี และรูปแบบ
- มีเครื่องมือคัตชนคลิปและ auto editor สำหรับช่วยเตรียมงานตัดต่อ

### 8. สตูดิโอเล่าเรื่อง

- ช่วยทำคลิปเล่าเรื่อง สาระ หรือสารคดี
- แบ่งเนื้อหาเป็น scene
- ช่วยสร้าง prompt, เสียง, SFX และ assets ที่เกี่ยวข้อง
- Export workflow สำหรับเอาไปทำคลิปต่อ

### 9. รันบอท Flow

- จัดคิว prompt สำหรับงาน bulk
- ช่วยรันงานซ้ำ ๆ ที่ต้องส่ง prompt หลายรายการ
- เหมาะกับ workflow ที่ต้องสร้างผลลัพธ์จำนวนมาก

### 10. Lazada Affiliate

- นำเข้ารายการสินค้าจากไฟล์ Excel
- ช่วยสร้างข้อความโปรโมตสินค้า
- เตรียม caption/prompt สำหรับคอนเทนต์ affiliate
- Export รายการพร้อมนำไปใช้งาน

### 11. Canvas Editor

- วางข้อความบนรูป
- ใส่ logo
- ปรับตำแหน่ง สี ขนาด และ effect ของข้อความ
- Export รูปสำหรับใช้ลงเพจ

### 12. จัดการ Assets

- เก็บและจัดการไฟล์เสียง เพลง ฟุตเทจ และไฟล์ประกอบงาน
- ใช้เป็นคลังกลางสำหรับ workflow ต่าง ๆ
- ช่วยให้หาของที่ใช้ประจำได้ง่ายขึ้น

### 13. Dashboard และติดตามงาน

- ดูภาพรวมงาน
- จัดการ todo/kanban
- เชื่อมข้อมูลกับ Google Sheets ได้ถ้าตั้งค่า API
- ใช้ติดตาม stock และสถานะงานในทีม

### 14. เครื่องมือระบบ

- เครื่องมือเคลียร์ไฟล์ขยะบางส่วน
- เครื่องมือสแกน/ตรวจสอบความปลอดภัยเบื้องต้น
- เหมาะสำหรับช่วยดูแล workspace ระหว่างทำงาน

## ข้อควรระวังเรื่องข้อมูลลับ

อย่าส่งไฟล์เหล่านี้ให้คนอื่นถ้ามีคีย์หรือข้อมูลส่วนตัว:

```text
public/app_data/api_profiles.json
.env
.env.local
```

โปรเจกต์นี้ตั้งค่า `.gitignore` ไว้แล้วว่าไม่ควร commit ไฟล์ API profile แต่ก่อนส่งต่อควรเช็คอีกครั้งด้วย:

```bash
git status --short
```

ถ้าเห็นไฟล์ที่มี token หรือ API key โผล่มาในรายการ ห้าม commit จนกว่าจะลบหรือ ignore ให้ถูกต้อง
