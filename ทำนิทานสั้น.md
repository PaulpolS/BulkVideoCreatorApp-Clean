# Project Name: AI Video Storyboard Architect

## Objective
Build a web-based tool for AI Video creators to generate high-fidelity Character Reference Sheet prompts and structured Storyboard prompts for Image-to-Video workflows.

## Tech Stack (Recommended)
- Framework: Next.js (App Router)
- Styling: Tailwind CSS
- Icons: Lucide React
- Language: TypeScript

## Features & UI Requirements

### 1. Navigation & Layout
- A clean, professional dashboard with a sidebar or tabs to switch between:
  - **Module 1: Character Reference Creator**
  - **Module 2: Storyboard Generator**

### 2. Module 1: Character Reference Sheet (Step 1)
**UI Elements:**
- Input: Textarea for "Describe your character" (e.g., "เป็ดใส่เสื้อฮู้ดสีเทา").
- Action: "Generate Ref Sheet Prompt" button.
- Output: A card displaying the generated English prompt with a "Copy to Clipboard" button.

**Logic:**
- Expand user input into a detailed professional 3D technical model turnaround prompt.
- **Strict Rule:** If multiple characters are detected, generate separate prompt blocks for each character to prevent concept bleeding.
- **Template to use:**
  "Create a professional character reference sheet of [Character Details]. The sheet uses a clean, neutral plain grey background with subtle grid lines and is presented as a high-fidelity 3D technical model turnaround... [Include details about Top Row: 4 views, Bottom Row: 3 portraits]."

### 3. Module 2: Storyboard Generator (Step 2)
**UI Elements:**
- Input: Textarea for "Story Context" (Thai).
- Input: Number of scenes (Numeric input).
- Action: "Build Storyboard" button.
- Output: A list of Scene Cards.

**Scene Card Structure (Logic for 1 Scene = 8 seconds):**
Each card must display:
1. **Thai Script/Dialogue:** บทพูดภาษาไทยที่กระชับและน่าสนใจ
2. **Start Image Prompt (0-4s):** English prompt focusing on action/environment. (Strictly NO physical character descriptions).
3. **End Image Prompt (4-8s):** English prompt showing the transition/result.
4. **Video Motion Prompt:** English instruction for camera movement and physics.

**Crucial Prompt Engineering Logic:**
- The AI must be instructed to **EXCLUDE** physical descriptions (clothes, hair, face) in the storyboard image prompts because the user will upload Character Reference images in the external video tool. Focus only on behavior, location, and camera.

### 4. Aesthetics & Branding
- High-end Management Consulting look (Clean, structured, white/grey/blue-indigo palette).
- Aesthetic typography using Google Fonts (Kanit for Thai, Inter for English).
